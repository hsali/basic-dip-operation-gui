#Basic Digital Image Processing Operation GUI app

##These type of operation this application perform

    1. Spatial Domain Operations
   
    * Rotating
    * Scaling
    * Shifting
   
    2. Intensity Transformation
    
    * Negative
    * Log
    * Gamma

    3. Basic Filters
 
    * Sharpening Filters
   
     * Sabel
     * Prewitt
     * Roberts
     * Canny
     * Laplacian
   
    * Smoothing Filters
     
     * Min
     * Max
     * Mean
     * Median
